# gui

Multi-purpose GUI made with QT and Python.

-----
### Usage 
Install the dependencies. SQLite3 should come pre-installed with Python 3.6 

```sh
$ pip3 install pyqt5
```
To run, simply run login.py in your terminal
```sh
$ python3 login.py
```

![](./storage/demo.gif)

-----
### Features 
* Enables user account creation and login.
* Utilizes SQLite
